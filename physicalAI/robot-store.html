Robot Store

A capability model for any embodiment.

URDF/USD tells you the geometry. Artha's capability layer answers the question that actually matters for training: what can this robot do — reach, payload, dexterity, sensing — and how does that transfer across bodies.

vs. Menagerie / Isaac asset zoos: those ship kinematics & dynamics. We add a queryable capability abstraction and cross-embodiment transfer — both unstandardized, both open research, both reusable across every world in the store.

Capability query

artha robots.query( task = "load dishwasher", world = "kitchen_counter_v3", ) # → candidates ranked by capability fit # Humanoid-G1 fit 0.91 ✓ reach ✓ payload ✓ bimanual # Bimanual YAM fit 0.58 ✗ reach (floor cabinet) # SO-ARM-101 fit 0.22 ✗ reach ✗ payload
The differentiator: match task × world × embodiment automatically, so a customer evaluating "which robot can do this job in this room" gets an answer before collecting a single real demo — and transfer learnings when they switch bodies.
